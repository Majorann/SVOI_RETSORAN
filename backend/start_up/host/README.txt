Как запустить хостинг из этой папки:

1) Дважды щелкните: 03_start_all.bat
   - Запускается локальное приложение (waitress на 127.0.0.1:8000)
   - Запускается общедоступный туннель (localhost.run)

2) В окне туннеля скопируйте URL, например:
   https://<random>.lhr.life

3) Оставьте оба окна открытыми, пока сайт не будет доступен онлайн.

Остановить все:
- Дважды щелкните: 04_stop_all.bat
Ручной режим:
- 01_start_app.bat      (только локальное приложение)
- 02_start_public_tunnel.bat (только интернет-туннель)
